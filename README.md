# Market Sentiment Analysis
## Bitcoin Market Sentiment vs Trader Performance Analysis

An exploratory data analysis project that examines the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid historical trading data.

This project explores the relationship between **Bitcoin market sentiment** and **trader performance** using two datasets:

- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trader Data

## Exploratory Data Analysis

The following analyses were performed:

- Market Sentiment Distribution
- Average Profit by Sentiment
- Total Profit Analysis
- Win Rate Analysis
- Buy vs Sell Analysis
- Trade Size Analysis
- Fee Analysis
- Top Trader Analysis
- Correlation Heatmap
- Daily Profit Trend
